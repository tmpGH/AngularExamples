There are two events: ngOnChanges and ngDoCheck. Theirs invocations are visible in console window.


The ngDoCheck happens when component's property changes, but also when content of component's property changes.
We need one basic type property (e.g. string) and one object with inner property.


string property value: {{ stringProperty }}
object's property value: {{ someClass.NumberProperty }}


The ngOnChanges take place when component's property decorated with @Input is changed.
We need one embeded component with @Input property.
