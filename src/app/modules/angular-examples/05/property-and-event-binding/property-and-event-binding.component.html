After binding an object by "moustache" binding: 

    ... {{ objectForBinding }} ...

, we can see something like below 

    {{objectForBinding}}

instead of the object value.


It's possible to bind a command with an event directly: 

    ... (click) = "numberVariable = numberVariable + numberVariable" ...

When we do an action:

Result (numberVariable) is: {{numberVariable}}


It's also possible to run change detection by event binding: 

    ... (click) = "0" ...

In component we can reach the html element through event.target .

The event.target.value is "{{targetValue}}".
