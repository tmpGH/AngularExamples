It's possible to bind an object with boolean properties, named as styles that we use, and control the element's style by setting values of object's properties.
In this example the object 

    styleClasses = {{'{'}}
        red: true,
        orange: false,
        yellow: false,
        green: false,
        blue: false,
        navy: false,
        violet: false
    };

is bound by ngClass directive: 

    ... [ngClass]="styleClasses" ...


place with changed color
