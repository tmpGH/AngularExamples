Simple use of async pipe.

In component: 

    numbers$: Observable<number> = interval(1000);

In template: 

    ... {{numbers$ | async}} ...

Result numbers (every {{milliseconds}} milliseconds): {{numbers$ | async}}


Async pipe with *ngFor.

In component: 

    this.numbers2$ = this.numbers$.pipe( take(10), toArray() );

In template: 

    ... *ngFor="let number of numbers2$ | async"> {{number}} ...

Result numbers (first {{countToTake}}):

  • {{number}}
