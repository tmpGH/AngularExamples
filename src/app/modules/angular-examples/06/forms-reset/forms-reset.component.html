We can clear form's fields (inputs) by calling NgForm.resetForm() method.
In component's code we should define reference to form: 

    @ViewChild('myForm') inputForm: NgForm;

and use mentioned method of this reference 

    this.inputForm.resetForm();
    // the same behavior
    // this.inputForm.reset();

Its done by button [ clear (by resetForm) ] . Button [ reset ] brings initial values back.
